## Accessing the Content Manager

1. Open your web browser (Chrome, Firefox, Safari, or Edge)
2. Go to: **https://dime73.github.io/ArtByJen/admin/**
3. You should see the "Content Manager" login screen

## Logging In

1. Click the "Login with GitHub" button
2. You'll be redirected to GitHub
3. Authorize the application (only needed the first time)
4. You'll be redirected back to the Content Manager

**Note**: You need to have access to the GitHub repository (Dime73/ArtByJen) to edit content.

## Updating Text Content

### Editing the Home Page Hero Section

1. In the Content Manager, click on **"Site Settings"** in the left sidebar
2. Click on **"Home Page Hero"**
3. You'll see two fields:
   - **Title**: The main heading on your home page
   - **Subtitle**: The text below the main heading
4. Make your changes
5. Click **"Save"** at the top
6. Click **"Publish"** to make your changes live

### Editing the About Section

1. Click on **"Site Settings"** → **"About Section"**
2. You'll see three fields:
   - **Section Title**: The "About the Artist" heading
   - **Paragraph 1**: Your first paragraph
   - **Paragraph 2**: Your second paragraph
3. Edit the text as needed
4. Click **"Save"** then **"Publish"**

### Editing Contact Information

1. Click on **"Site Settings"** → **"Contact Information"**
2. Update:
   - **Section Title**: The heading for the contact section
   - **Description**: The introductory text
   - **Email**: Your contact email address
3. Click **"Save"** then **"Publish"**

## Managing Gallery Items

### Editing an Existing Gallery Item

1. Click on **"Gallery Items"** in the left sidebar
2. You'll see a list of all your artwork
3. Click on the item you want to edit
4. You can change:
   - **Title**: The name of the artwork
   - **Medium/Description**: e.g., "Oil on Canvas, 2024"
   - **Image**: Click to upload a new image (see below)
   - **Alt Text**: Description for accessibility
   - **Order**: Change the number to reorder items (lower numbers appear first)
5. Click **"Save"** then **"Publish"**

### Adding a New Gallery Item

1. Click on **"Gallery Items"**
2. Click the **"New Gallery Item"** button
3. Fill in all the fields:
   - **Title**: Name your artwork
   - **Medium/Description**: Describe the medium and year
   - **Image**: Click to upload your image
   - **Alt Text**: Describe what's in the image
   - **Order**: Choose where it should appear (e.g., 10 for tenth position)
4. Click **"Save"** then **"Publish"**

### Deleting a Gallery Item

1. Click on **"Gallery Items"**
2. Click on the item you want to delete
3. Click the **"Delete"** button (usually near the top)
4. Confirm the deletion
5. Click **"Publish"** to apply the change

## Uploading New Images

When you see an **Image** field:

1. Click on the image field or the "Choose an image" button
2. You'll see two options:
   - **Upload**: Upload a new image from your computer
   - **Library**: Choose from previously uploaded images
3. To upload a new image:
   - Click **"Upload"**
   - Click **"Choose files"**
   - Select an image from your computer
   - Click **"Insert from Media Library"**

**Image Tips**:
- Use JPG or PNG format
- Recommended size: 800-1200 pixels wide for gallery items
- Keep file sizes under 2MB for faster loading
- Use descriptive file names (e.g., "sunset-painting.jpg")

## Publishing Your Changes

**Important**: Your changes are not live until you publish them!

1. After making any changes and clicking **"Save"**
2. Look for the **"Publish"** button at the top of the page
3. Click **"Publish"**
4. A confirmation message will appear
5. Your changes will be live on the website within 1-2 minutes

**Note**: Publishing creates a new version in GitHub, so you can always undo changes if needed.

## Viewing Your Changes

After publishing:

1. Wait 1-2 minutes for the changes to deploy
2. Open a new browser tab
3. Go to: **https://dime73.github.io/ArtByJen/**
4. Refresh the page (press F5 or Ctrl+R / Cmd+R)
5. You should see your updated content

**Tip**: If you don't see changes, try a "hard refresh":
- Windows/Linux: Ctrl + Shift + R
- Mac: Cmd + Shift + R

## Troubleshooting

### "I can't log in"
- Make sure you have access to the GitHub repository
- Try logging out of GitHub and logging back in
- Clear your browser cache and try again

### "My changes aren't showing up"
- Make sure you clicked both "Save" AND "Publish"
- Wait 2-3 minutes for changes to deploy
- Try a hard refresh on your website (Ctrl+Shift+R or Cmd+Shift+R)
- Check if you're looking at the correct website URL

### "I made a mistake and want to undo it"
- Don't worry! All changes are tracked in GitHub
- Contact your website administrator to restore a previous version
- Or, simply go back into the Content Manager and fix the mistake

### "Images aren't uploading"
- Check the file size (should be under 2MB)
- Make sure the file is JPG or PNG format
- Try a different browser
- Try refreshing the page and uploading again

## Need More Help?

If you encounter any issues not covered here:

1. Take a screenshot of the error message
2. Note exactly what you were trying to do
3. Contact your website administrator with this information
